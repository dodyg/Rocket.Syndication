# Code Style

- Follow C# coding convention at https://learn.microsoft.com/en-us/dotnet/csharp/fundamentals/coding-style/coding-conventions

# C# Project Instructions

**Build Command:** `dotnet build`
**Run Command:** `dotnet run --project WebApp`
**Test Command:** `dotnet test`

Ensure that after any code changes, you run the `dotnet format` command for consistent style, followed by `dotnet build` and `dotnet test` to verify correctness.
